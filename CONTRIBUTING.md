# Contributing to smartmeter

Thank you for your interest in contributing! This is an academic embedded systems project by Yaswanth Chlliboina. Contributions, suggestions, and improvements are welcome.

## How to Contribute

### Reporting Issues
- Open an issue describing the problem clearly
- Include hardware setup, error messages, and steps to reproduce

### Suggesting Improvements
- Fork the repository
- Create a new branch: `git checkout -b feature/your-feature`
- Commit changes: `git commit -m "Add: your feature description"`
- Push and open a Pull Request

### Code Style (Embedded C)
- Follow STM32 HAL naming conventions
- Comment all hardware-specific values (pin numbers, baud rates, timing)
- Test on STM32 F446RE Nucleo board before submitting

## Running the Simulator
No hardware needed:
```bash
cd simulator
python simulate.py
```
